# Django Croissant Clicker

This is my first django application which acts as a simple cookie clicker generated with django.

## Instructions for Build and Use

Steps to build and/or run the software:

1. Open CMD and navigate to the project directory with the cd command (cd "your/directory/here")
2. Run the following command: python manage.py runserver
3. Copy the http link listed after "Starting development server at ..." and paste it into your browser
4. at the end of the link add a slash and then "counter" — the complete link should look something like this: http://127.0.0.1:8000/counter

Instructions for using the software:

1. After following the above steps, you should see a page appear with welcome text.
2. Click the button to increase your croissant count.
